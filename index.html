Color-Changing Clock

Created On: May 18, 2016

This color-changing clock is controled by JavaScript.

This application is based on the tutorial JavaScript and the DOM (Part 2 of 2) put out by the DevTips YouTube channel.

In this tutorial, viewers were taught how to create this color-changing clock, and then display the hours, minutes, and seconds (in military time), as a color hex code. For example, 2:27:46pm (which is 14:27:46 in military time) would display in the application as "#142746", and produce a deep blue color. This clock was originally inteded to cover an entire page, with the application itself loading onto the body, instead of a seperate div.

For my version of this application, I modified the JavaScript so that not only would the time be displayed, but it would be displayed as a 12-hour clock, instead of 24-hour military time. Since this is a 12-hour clock, I also added functionality to include either "am" or "pm" at the end of the time, depending on the time of day the clock is viewed.

Update - May 20, 2016:

After a day of this appication being live, and reveiving feedback on it, I've decided to make a few minor changes. The first is that, I have decided to change the "am" or "pm" designation from lowercase to uppercase, to make the entire line of text appear more uniform, and the second change is that I've decided to add the hex color beneith the time, so that viewers won't have to convert numbers in their head.

Additionally, I've modified the JavaScript so that instead of adding the text via "textContent" it will be added via "innerHTML". This has allowed me to be able to specify that the time should be wrapped in an H2 tag with the class of "time", and the hex color code wrapped in an H3 tag, with the class of "hex-code". This change has allowed me to each individually, with CSS.
